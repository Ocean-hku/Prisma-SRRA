# 【棱镜 Prisma】产品需求文档 (PRD)

## 1. 产品概述
**产品名称**：棱镜 Prisma
**产品定位**：下一代高维动态自我认知导航系统（网页版）
**核心理念**：打破传统 MBTI/SBTI 的硬边界与静态标签，结合心理学（特质维度）、社会学（拟剧论）与哲学（存在主义），为用户提供动态、高维、具柔性边界的自我认知体验。

## 2. 目标用户
*   **厌倦标签化的人群**：对现有的“抽屉式”性格测试（如 16 型人格）感到局限，渴望更精准自我认知的年轻人。
*   **寻求深度成长的探索者**：不满足于简单的性格描述，希望了解自身行为背后的防御机制并获得改进建议的人。
*   **泛心理学/哲学爱好者**：对自我探索、存在主义、社会学拟剧论等概念感兴趣的高认知群体。

## 3. 核心功能需求

### 3.1 沉浸式情境测验 (The Assessment)
*   **交互形式**：全屏沉浸式卡片问答，摒弃传统枯燥的单选列表。
*   **题目设计**：采用“情境变量”设计（如：在高压职场下 vs 独处深夜），探索用户在不同社会场域下的反应。
*   **动态感**：题目切换时具备平滑的过渡动画，营造深邃、内省的氛围。

### 3.2 3D 性格星图与柔性边界 (The Map of Self)
*   **视觉呈现**：测试完成后，不直接给出一个死板的四字标签，而是生成一个动态的、极具科技感的 3D 星空图（或高维坐标可视化）。
*   **数据表达**：
    *   显示用户所在的“主星云”（主要倾向类型）。
    *   显示**柔性边界（Soft Margin）**：通过距离和连线，展示用户距离其他相邻类型的欧氏距离（例如：“您偏向深思者，但距离行动派仅有 0.25 的距离”）。

### 3.3 存在主义诊断与微小行为实验 (The Prescription)
*   **诊断报告**：以“存在主义病历”的形式，犀利指出用户当前性格特质背后的防御机制。
*   **行动处方**：提供 1-3 个具体、微小的行为实验建议，鼓励用户打破舒适区，实践亚里士多德的“中道”。
*   **分享卡片**：支持生成具有极高审美质感（如玻璃拟物风、镭射风）的结果卡片，供用户分享至社交媒体（充当高级“社交货币”）。

## 4. 审美与设计方向 (Aesthetic Direction)
*   **基调**：**深邃 (Deep) / 内省 (Introspective) / 科技哲学 (Tech-Philosophical)**。
*   **色彩体系**：以深色模式 (Dark Mode) 为主，背景采用深渊黑、午夜蓝，辅以发光的镭射紫、霓虹蓝、星云粉作为点缀色。
*   **排版与字体**：
    *   标题字体：选用极具几何感或衬线感的优雅字体（如 Playfair Display, Cinzel, 或现代的 Space Grotesk）。
    *   正文字体：清晰易读的无衬线字体。
    *   大量留白，营造呼吸感与高级感。
*   **动效 (Motion)**：缓慢的呼吸光效、粒状噪点背景 (Grain texture)、卡片悬浮时的玻璃折射效果 (Glassmorphism)，以及星图的平缓自转。

## 5. 最小可行性产品 (MVP) 页面结构
1.  **Landing Page (首页)**：
    *   极简的 Slogan ：“你不是一个标签，你是一片星云。”
    *   一个脉吸的“开始探索”按钮。
    *   背景为缓慢流动的暗色渐变或噪点纹理。
2.  **Assessment Page (测试页)**：
    *   全屏显示的题目卡片。
    *   沉浸式的进度条（如光带填充）。
3.  **Result Page (结果页)**：
    *   **首屏**：震撼的 3D 星图动效（或 2D 拟 3D 散点图），标出用户的坐标。
    *   **下滚**：解读面板，包含“主轴倾向”、“柔性边界分析”、“存在主义诊断”。
    *   **底部**：“微小行为实验”行动卡片与“生成分享图”按钮。

## 6. 非功能性需求
*   **响应式设计**：必须完美适配移动端（优先）和桌面端。
*   **性能**：动效必须流畅，避免在低端设备上卡顿。
*   **无障碍**：足够的色彩对比度。
